# dotfiles

my thinkpad T470 dotfiles.

## Installation

```bash
git clone https://github.com/moxi-git/dotfiles.git -b "dotfiles"
cd "dotfiles"
./install.sh
```

> [!NOTE]
> this dotfiles work best with Arch since they made with Arch linux in mind. other distros maybe can work too
